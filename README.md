# ft_ping

## Prerequisites

- [Vagrant](https://www.vagrantup.com/)
- [Virtual Box](https://www.virtualbox.org/)

## Usage

### Virtual Machine

```sh
# Start the Virtual Machine
make vm.up
# Connect through SSH
make vm.ssh
# Open wireshark with X11 forwarding
wireshark
```

You can log in as `vagrant:vagrant`
